# godot-skills

Experimental notes and guidelines for working with Godot.

## Structure

* `main-godot-skill.md` — General concepts and shared practices
* `godot-2d-specifics.md` — Notes specific to 2D development
* `godot-3d-specifics.md` — Notes specific to 3D development

## Purpose

This repository serves as a lightweight reference for organizing knowledge about Godot, separating common concepts from 2D and 3D-specific considerations.

Content is exploratory and may evolve or change over time.
